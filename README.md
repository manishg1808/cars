# Luxury Fleet Services

Frontend-only multi-page transportation website for Orlando luxury ride presentation.

This repository contains the UI layer only:

- HTML pages
- CSS styling
- JavaScript interactions
- local media assets

Backend PHP files and `PHPMailer` are intentionally excluded.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Remix Icon CDN

## Included Files

- `index.html`
- `services.html`
- `fleet.html`
- `gallery.html`
- `contact.html`
- `request-quote.html`
- `login.html`
- `styles.css`
- `script.js`
- `assets/`

## Notes

- Forms in this repo are UI demos only.
- Frontend submit actions are handled in JavaScript and do not send data to a server.
- Local images and video are loaded from `assets/`.

## Run Locally

Open `index.html` in a browser, or use Live Server in VS Code.

## Project Focus

- premium transportation branding
- responsive layouts
- fleet showcase
- gallery experience
- booking and enquiry UI
- luxury visual presentation across desktop and mobile
