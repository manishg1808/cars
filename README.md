# cars

## Luxury Fleet Services

Simple multi-page static website for premium car transportation services built with HTML, CSS, and JavaScript.

## Project Stack
- HTML5
- CSS3
- Vanilla JavaScript
- Remix Icon CDN

## Pages
- `index.html` - Home page with hero video, service storytelling, booking request, reviews, CTA
- `services.html` - Service details and rates tables
- `fleet.html` - Fleet listing with vehicle specs and booking actions
- `gallery.html` - Gallery grid with image popup/lightbox
- `contact.html` - Contact information and contact form
- `request-quote.html` - Standalone quote page (legacy)
- `login.html` - Login page
- `register.html` - Register page

## Main Features Implemented
- Fixed navbar with scroll-based background effect
- Mobile menu toggle for smaller screens
- Home hero video banner with end-image fallback
- Dynamic hero heading text animation
- Gradient button styles and blue/white theme
- Home quick enquiry form
- Home service scrollytelling section
- Why Choose section redesign with icons
- How It Works timeline section redesign
- Home Quick Booking Request section redesign with 70/30 layout
- Google review marquee section
- Gallery hover captions and click-to-open popup preview
- Fleet and gallery grid layout updates
- Services page redesign sections (Orlando Car Services, Why Choose, How It Works)
- Rates section placed directly below services banner
- Responsive improvements across sections and rates tables
- Navbar login icon support
- CTA sections with gradient styling
- Request Quote popup modal with full fields and no page navigation

## Assets
All site media is referenced from local `assets/` (plus existing CDN icon/image links where already present).

## Run Locally
Open `index.html` directly in browser, or use Live Server in VS Code.
